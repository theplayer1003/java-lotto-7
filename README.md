# 시스템 책임 파악

1. 구매하기
   1. 구입 금액을 입력 받는다.
   2. 돈 객체를 생성한다.
   3. 구매자 객체를 생성한다.
   4. 로또를 구매한다.

2. 발행하기 
   1. 일반 로또 티켓 발행한다.
      1. 무작위 숫자 6자리
   2. 당첨 로또 티켓 발행한다.
      1. 입력 받은 숫자 6자리 파싱, 보너스 번호 1자리
    
3. 당첨 확인하기
   1. 당첨표를 생성한다.
   2. 수익률을 계산한다.

# 적절한 객체 또는 역할을 찾아 책임을 할당
